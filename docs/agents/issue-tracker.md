# Issue tracker: GitHub

Issues and PRDs live in GitHub Issues at `xleepy/wrist-menu`. Use the `gh` CLI for operations. Pull requests are not a triage request surface.

## Wayfinding operations

- The map is an issue labelled `wayfinder:map`.
- Tickets are GitHub sub-issues labelled `wayfinder:research`, `wayfinder:prototype`, `wayfinder:grilling`, or `wayfinder:task`.
- Use native GitHub issue dependencies where available.
- Claim a ticket by assigning it before beginning work.
- Resolve a ticket by commenting with the answer, closing it, and adding its linked gist to the map's Decisions-so-far section.
- If sub-issues or dependencies are unavailable, use task-list links and explicit `Blocked by:` lines.
